# 🏥 Clínica Vital Salud - Sistema de Gestión Médica

¡Bienvenido! Este es un sistema web completo desarrollado en PHP para la gestión y control de citas médicas en la Clínica Vital Salud. Está diseñado de forma ordenada usando la estructura MVC (Modelo-Vista-Controlador).

## 🚀 Características Principales
* **Módulo de Citas:** Registro, edición y control de agendas médicas.
* **Control de Usuarios:** Roles para administradores, médicos y pacientes.
* **Panel de Control (Dashboard):** Estadísticas y reportes de la clínica en tiempo real.

## 🛠️ Tecnologías Usadas
* **Backend:** PHP (Estructura limpia con MVC)
* **Frontend:** JavaScript, CSS, SCSS, Bootstrap
* **Base de Datos:** MySQL / MariaDB

## 📦 Cómo Ejecutar el Proyecto Localmente

1. Clona este repositorio o descarga el código.
2. Mueve la carpeta a tu servidor local (por ejemplo, dentro de `c_xampp/htdocs/`).
3. Importa el archivo de la base de datos (por ejemplo: `clinicavitalsalud.sql`) en tu phpMyAdmin.
4. Abre tu navegador e ingresa a `http://localhost/clinicavitalsalud`.
